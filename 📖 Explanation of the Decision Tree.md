# 📖 Explanation of the Decision Tree  
This decision tree is **deterministic**, meaning:  
* Each question has **fixed outcomes**  
* Same input always leads to the **same decision path**  
  
## 🔹 Step 1: Activity Check  
The process begins by checking whether the student studied or worked.  
* If **No** → The system identifies the reason:  
    * Valid → Recovery plan  
    * Invalid → Discipline improvement  
* If **Yes** → Move to deeper reflection  
  
## 🔹 Step 2: Topic Identification  
The student explicitly defines **what they worked on**, ensuring clarity instead of vague reflection.  
  
## 🔹 Step 3: Understanding Evaluation  
The student evaluates their understanding:  
* If **fully understood**:  
    * Check if they can explain it → ensures real understanding  
* If **not understood**:  
    * Identify the exact issue:  
        * Concept gap  
        * Lack of practice  
        * Logical confusion  
**Step 4: Practice Assessment**  
Learning without practice is incomplete.  
* If practiced:  
    * Success → Increase difficulty  
    * Failure → Analyze mistakes  
* If not practiced:  
    * Add practice to next plan
